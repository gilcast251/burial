1. Esta es la pagina web de mi proyecto final del curso de EDTecnica, es una pagina que realizé para la empresa donde trabaje que se especializa en trabajos de Granito, Mármoles y Cuarzo.

2. Cuenta con una presentacion minimalista, ubicación en GoogleMaps, funcionalidad de correo y modo Oscuro.

3. Tambien cuenta con una aplicación interna que permite calcular una cotización rapida para los usuarios basado en los datos requeridos (Tipo de Material, Ubicacion de la Obra, etc...).

4. La aplicacion trabaja con 2 API, la primera es JSPDF que permite mostrar los resultados de las operaciones logicas y la cotización completa un archivo PDF que se le descargara al usuario, el formato de la cotizacion se logra gracias a herramientas parecidas a HTML Canvas con coordenadas X y Y.

5. La segunda API es la de Dolartoday para calcular el precio del dolar paralelo de manera automatica (se utiliza esta referencia solo como ejemplo).

6. Tiene codigo adicional de javascript para slider de imagenes.

7. Tambien tiene un codigo para cotizacion en Dolares Americanos, aunque le hacen falta adaptaciones al documento PDF.
